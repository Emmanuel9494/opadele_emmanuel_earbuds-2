## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn package manager
- Modern web browser

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/trivox-earbuds.git
   ```
2. Navigate to the project directory:
   ```bash
   cd trivox-earbuds
   ```
3. Install dependencies:
   ```bash
   npm install
   # or
   yarn install
   ```

### Running the Project
```bash
npm run dev
# or
yarn dev
```
